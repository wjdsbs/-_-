# 카페

* 그리다디저트

* 인천광역시 연수구 송도동 8-3 휴먼빌파크 1층 108호

유럽위 노천카페가 즐비한 거리와 비슷한 분위기의 화이트톤 깔급하고 모던한 인테리어가 돋보이는 공간.
르꼬르동 불르 출신의 수석 파티시에가 매일 매장에서 모든 디저트를 직접 만들고 특별주문을 제외하고는 모든 디저트에 합성첨가물을 사용하지 않는다.
